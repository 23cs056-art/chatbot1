🔹 ARTIFICIAL INTELLIGENCE (AI)
Artificial Intelligence (AI) is a field of computer science that focuses on creating machines and systems that can perform tasks which normally require human intelligence. These tasks include learning from experience, understanding information, solving problems, making decisions, and recognizing patterns.
AI systems work by using large amounts of data and applying algorithms to analyze that data. Over time, they improve their performance by learning from past results. Unlike traditional programming, AI systems can adapt and make better decisions as more data becomes available.

Types of AI:
Narrow AI: Designed for specific tasks such as chatbots or recommendation systems.
General AI: Aims to perform any human-level task (still under development).
Super AI: A theoretical concept where machines surpass human intelligence.

Applications:
AI is widely used in healthcare for disease prediction, in finance for fraud detection, in e-commerce for product recommendations, and in transportation for self-driving cars.

Limitations:
AI requires large amounts of data, may produce biased results, and lacks human emotions and creativity.

Example:
Netflix uses AI to recommend movies based on your viewing history.

🔹 MACHINE LEARNING (ML)
Machine Learning (ML) is a subset of Artificial Intelligence that allows machines to learn from data without being explicitly programmed. Instead of following fixed instructions, ML systems identify patterns in data and use those patterns to make predictions or decisions.
ML models are trained using datasets. During training, the system learns relationships between inputs and outputs. Once trained, the model can predict outcomes for new data. The more data it receives, the better it performs.

Types of Machine Learning:
Supervised Learning: Uses labeled data to learn (e.g., spam detection).
Unsupervised Learning: Finds patterns in unlabeled data (e.g., clustering).
Reinforcement Learning: Learns through rewards and punishments.

Applications:
ML is used in recommendation systems, fraud detection, speech recognition, and image classification.

Limitations:
ML requires large datasets, can suffer from overfitting, and may produce incorrect predictions if data is poor.

Example:
Email services use ML to filter spam messages automatically.

🔹 DEEP LEARNING (DL)
Deep Learning is a specialized area of Machine Learning that uses neural networks with multiple layers to process data and learn complex patterns. It is inspired by the structure of the human brain.
Deep learning models automatically extract features from data, which reduces the need for manual feature engineering. These models are particularly useful for tasks involving large amounts of unstructured data such as images, audio, and text.

Types:
CNN (Convolutional Neural Network): Used for image processing.
RNN (Recurrent Neural Network): Used for sequence data like text and speech.

Applications:
Deep learning is used in face recognition, voice assistants, self-driving cars, and medical image analysis.

Limitations:
It requires high computational power, large datasets, and is difficult to interpret.

Example:
Face unlock systems in smartphones use deep learning to recognize users.

🔹 NATURAL LANGUAGE PROCESSING (NLP)
Natural Language Processing (NLP) is a branch of AI that enables machines to understand, interpret, and respond to human language. It bridges the gap between human communication and computer understanding.NLP works by breaking text into smaller units (tokens), analyzing grammar and meaning, and using models to understand context. It allows machines to process both written and spoken language.

Applications:
Chatbots and virtual assistants
Language translation
Sentiment analysis

Limitations:
NLP struggles with understanding context, sarcasm, and ambiguous language.

Example:
Google Translate converts text from one language to another using NLP.

🔹 LARGE LANGUAGE MODELS (LLM)
Large Language Models (LLMs) are advanced AI models trained on massive amounts of text data to understand and generate human-like language. They are capable of answering questions, generating content, and assisting users in conversations.
LLMs work by predicting the next word in a sentence based on patterns learned from training data. They use transformer architecture and attention mechanisms to understand context.

Applications:
Chatbots like ChatGPT
Content writing
Code generation

Limitations:
They may generate incorrect or misleading information and require high computational resources.

Example:
ChatGPT answering user queries.
